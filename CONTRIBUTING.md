# How to contribute

## Ways to contribute

1. Tackle [an open issue](https://github.com/devtees/shopify-product-filter/issues) and [submit a pull request](#submit-a-pull-request)
2. Try the code out on a test shop and [report any issues you find](https://github.com/devtees/shopify-product-filter/issues/new)
3. Let us know if there are any [features or improvements you'd like to see](https://github.com/devtees/shopify-product-filter/issues)
5. Help improve [the documentation](https://github.com/devtees/shopify-product-filter/blob/master/README.md)

## Submit a pull request

Want to propose a change? Great. We'd love your help. Here's how:

1. Fork the project
2. Create a descriptively named feature branch
3. Commit your changes
4. Submit a pull request

For more information see [GitHub flow](https://guides.github.com/introduction/flow/) and [Contributing to Open Source](https://guides.github.com/activities/contributing-to-open-source/)
